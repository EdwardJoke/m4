## [v0.1.1] - 2026-05-25

### Added
- Compiler support for `for` loops inside function bodies (`compileForToChunk`)
- Compiler support for `index` expressions (array/vector indexing like `nums[0]`) in both `compileExpr` and `compileExprToChunk`
- Compiler support for `struct_lit` and field access expressions in `compileExprToChunk`, enabling struct construction and field access inside function bodies
- `try` expression compilation support in `compileExprToChunk`
- Scanner parenthesis-depth tracking to suppress indentation/dedent tokens inside parenthesized expressions, fixing multi-line struct literal parsing
- Parser `skipNewlinesAndIndent` utility for robust whitespace handling inside struct literals
- `README.md` with project overview, quick start guide, CLI usage, language reference, and project structure
- `LICENSE` (MIT) file
- 10 example Maple programs in `examples/`: hello_world, fizzbuzz, fibonacci, types_and_variables, structs_and_methods, control_flow, vectors_and_iteration, math_and_operators, error_handling

### Fixed
- Function argument register allocation in `compileExprToChunk` — arguments are now correctly moved to the register positions expected by the VM's `call` instruction (`cr + 1 + i`), fixing issues where recursive calls and chained expressions produced wrong results
- Struct literal parsing across multiple lines — scanner no longer emits erroneous indent/dedent tokens inside parentheses
- Removed duplicate `compileExprToChunk` function definition that caused compilation ambiguity
