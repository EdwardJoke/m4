# Changelog

## [v0.1.3] - 2026-05-26

### Added
- String comparison operations (`<`, `>`, `<=`, `>=`) with lexicographic ordering via `std.mem.order`
- String length support via the `len()` builtin on string values
- String character indexing via `s[i]` syntax, returning a `.char` value
- New example programs: `tests/string_test.m4`, `tests/string_ops.m4`

### Changed
- Refactored VM comparison logic: `cmpSlow` decomposed into `cmpSlow` → `cmpOrder` → `cmpValues` for cleaner architecture
- Updated error descriptions (r005, r008, r009) to document string support alongside vectors
- Improved error message text: "cannot compare non-numeric types" → "cannot compare values of different types"
- Added `--format` / `-f` flag to CLI help text for discoverability

### Fixed
- Help text now lists the `--format` flag which was implemented but undocumented
